Retail Bussiness Sales Analysis Project

introducion
this is dataset download for Kaggle website. This project focus on analyzing Retail Business Performance Analysis Dataset  using python libraries like (pandas,numpy,matplotlib and seaborn) and Create attractive visualization.In this data have four different table like product,customer,sales and store.
EDA Process
--Analyzed retail sales data by combining Customer, Product, Store, and Sales datasets.
-Performed data cleaning, handling missing values, removing duplicates, and data transformation to improve data quality.
-Conducted Exploratory Data Analysis (EDA) to identify sales trends, customer purchasing behavior, and product performance.
-Created visualizations to analyze revenue, quantity sold, top-performing products, and store-wise sales performance.
-Developed an interactive Power BI dashboard with KPIs, filters, and charts for business insights.
-Generated actionable insights to support data-driven decision-making and improve sales performance.
-Key Achievement: Improved understanding of sales patterns and identified top-selling products, profitable categories, and high-performing stores through data analysis and visualization.
